# 教育コンテンツ品質基準

## 語彙選択基準

### 例語の選定
1. **日常性**: 小学生が日常で触れる語彙
2. **頻度**: 使用頻度の高い語彙を優先
3. **適切性**: 年齢相応の内容
4. **多様性**: 音読み・訓読み両方をカバー

### 避けるべき語彙
- 専門用語（医学、法律など）
- 古語・文語
- 俗語・スラング
- 暴力的・不適切な内容を含む語

## 例文の品質

### 良い例文の条件
- 自然な日本語
- 文脈が明確
- 対象漢字の意味が伝わる
- 小学生が理解できる内容

### 例文パターン
```
「〜が〜する。」  # 基本形
「〜を〜する。」  # 動作対象
「〜で〜する。」  # 場所・手段
```

## 読みの正確性

### 音読み
- カタカナで表記
- 常用漢字表に準拠
- 小学校で習う読みに限定

### 訓読み
- ひらがなで表記
- 送りがなを含まない形
- 活用形は別途okuriganaExamplesで対応

## データ整合性

### 例語チェック
- 対象漢字が例語に含まれるか
- 読みが正しいか
- 重複がないか

### 対義語/類義語チェック
- 同学年内に存在するか
- 意味的に正しいペアか
- 双方向に設定されているか

## 品質指標

| 指標 | 目標値 |
|------|--------|
| 例語カバレッジ | 100%（全漢字に最低2例語） |
| 例文カバレッジ | 100%（全漢字に最低1例文） |
| 送りがなカバレッジ | 該当漢字の80%以上 |
| 対義語カバレッジ | 該当漢字の50%以上 |
| 部首カバレッジ | 50%以上 |
