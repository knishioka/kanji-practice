# 学習理論に基づく設計ルール

## 学習段階モデル

小学校での漢字学習は以下の段階で進行：

```
1. 構造理解 (strokeCount)
   ↓
2. 認識 (reading)
   ↓
3. 産出 (writing)
   ↓
4. 応用 (sentence)
```

各モードはこの学習段階に対応して設計されている。

## 認知的負荷理論

### 学年別配慮
| 学年 | 配慮事項 |
|------|----------|
| 1-2年 | 視覚的な手がかり多め、シンプルな例語 |
| 3-4年 | 音訓両方の習熟、熟語の導入 |
| 5-6年 | 抽象的な語彙、文脈での使用 |

### 問題設計
- 1問題あたりの情報量を制限
- ヒント表示のオプション提供
- 練習列数の調整可能

## 分散学習効果

### 実装方針
- ランダム出題オプション
- 同じ漢字を複数ページにまたがって出題
- 例語ベース生成で同じ漢字を異なる読みで出題

### 効果的な繰り返し
```
1回目: 新規学習
2回目: 1日後に復習
3回目: 1週間後に復習
4回目: 1ヶ月後に復習
```

## 間隔反復パターン

問題生成時の考慮：
- 新出漢字と既習漢字の混在
- 難度の段階的上昇
- 成功体験を積めるバランス

## 漢検対応

| 漢検級 | 対象学年 | 漢字数 |
|--------|----------|--------|
| 10級 | 1年生 | 80字 |
| 9級 | 2年生 | 160字 |
| 8級 | 3年生 | 200字 |
| 7級 | 4年生 | 202字 |
| 6級 | 5年生 | 193字 |
| 5級 | 6年生 | 191字 |

### 漢検対策モード
- 画数問題: 漢検で頻出
- 送りがな: 漢検の主要出題形式
- 対義語/類義語: 漢検の主要出題形式
