# 全般ルール

## 言語

- コードコメント: 教育コンテキストのため日本語推奨
- 変数名: 英語（camelCase）
- コミットメッセージ: 日本語または英語（PR内で一貫性を保つ）

## ファイル構成

- 1ファイル1コンポーネント
- テストは `__tests__/` ディレクトリに配置
- 定数は `constants/` ディレクトリに配置

## インポート

- `src/` からの絶対インポートを使用
- インポート順序: React、外部ライブラリ、内部モジュール、型
- Biomeが自動でインポートを整理

## エラーハンドリング

- null/undefined のケースは必ず処理
- TypeScript の strict mode チェックを活用
- ガード節には早期リターンを使用
